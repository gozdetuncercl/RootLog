# RootLog

AI-powered observability and root cause analysis platform.

RootLog helps developers analyze logs, detect anomalies, and identify possible root causes using AI-assisted insights.

## Planned Features

- Log ingestion
- AI-powered incident summaries
- Error clustering
- Root cause suggestions
- Search and filtering
- Dashboard
- Docker support

## Tech Stack

### Backend
- .NET 8
- PostgreSQL
- OpenAI API

### Frontend
- React + TypeScript

### Infrastructure
- Docker
